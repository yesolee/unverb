## 미션 시스템

미션은 두 가지 카테고리로 나뉜다.

### 🔍 관찰 미션 (경로 A) — 이미 하는 것 속에서 발견하기

일상 속 무의식적 행동의 의미를 발견하는 미션.

예시:
- "오늘 가장 오래 머무른 장소에서 사진 찍기"
- "점심에 뭘 먹을지 고를 때, 첫 번째로 떠오른 것 vs 실제 선택한 것 비교해보기"
- "하루 중 가장 편안했던 순간을 떠올려보기"

### 🌱 탐색 미션 (경로 B) — 작은 새로운 경험 해보기

새로운 자극에 대한 반응에서 나를 발견하는 미션.

조건: **돈 안 들거나 소액, 30분 이내, 집 근처에서 가능**

예시:
- "평소 안 가는 길로 출퇴근해보기"
- "편의점에서 한 번도 안 먹어본 거 하나 사먹기"
- "카페 대신 공원 벤치에서 10분 앉아보기"

### 공통 구조

```
미션 = 행동 지시 + 의미(왜 하는지) + 출처(논문 DOI)
```

예시:
- 행동: "카페 대신 공원 벤치에서 10분 앉아보기"
- 의미: "같은 '혼자이기'인데, 장소가 바뀌면 느낌이 어떻게 달라질까?"
- 출처: [1] Place Attachment and Identity... (DOI: 10.1037/xxx)

MVP 기준: 관찰 미션 15개 + 탐색 미션 15개 + 성찰 질문 30개

## AI 담임선생님 피드백

기록 저장 후 AI가 따뜻한 피드백을 제공한다.

**톤**: 초등학교 담임선생님이 빨간 펜으로 적어주는 느낌

**구조**: 공감 → 발견 포인트 → 다음 단계 힌트

**제약**:
- Layer 1 DB에 없는 내용은 생성 금지
- 금지 표현: 진단("당신은 우울증입니다"), 의료 조언("약을 먹어보세요"), 근거 없는 낙관("다 괜찮아질 거예요"), 감정 축소("그건 별거 아니야")

## 한 주 정리 (주간 리뷰) — 매주 일요일

### Part A: AI 자동 분석 (보기만 하면 됨)
- 타임라인: 이번 주 기록 시간순 나열
- 감정 그래프: 기록 속 감정 변화
- 키워드 클러스터: 자주 쓴 단어들
- 동사 발견 리포트: "이번 주 당신에게서 발견된 동사는..."

### Part B-1: Best Moment 뽑기 (~30초)
- 6일간의 기록 카드 6장 나열
- 가장 기억에 남는 1장 탭
- 선택적으로 "왜?" 한 줄 입력

### Part B-2: 동사 확인 + 다음 주 (~1분)
- AI가 찾은 이번 주 동사 확인 ("이번 주 당신은 '혼자이기'를 좋아하는 사람이었어요")
- "맞아" / "좀 다른데" 선택
- 다음 주 방향 2~3개 중 선택 ("더 탐색할래" / "다른 영역 해볼래" / "쉬어갈래")

### Closing: AI 담임선생님 주간 요약
- 한 주 전체를 아우르는 따뜻한 피드백

**전체 소요시간: ~2분**

## 출처 시스템

- 미션/질문/의미 텍스트에 [1][2] 각주 표시
- 탭하면 팝업으로 논문 제목/저자/DOI 확인
- 별도 출처 인덱스 페이지

## 대시보드 (P1)

- 발견한 동사 목록 ("'혼자이기'를 좋아하는 사람")
- 월별 미션 완료율
- 관찰 미션 vs 탐색 미션 반응 차이
