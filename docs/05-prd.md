# 05. PRD (Product Requirements Document)

**버전**: v1.1 (MVP) — 미션 카테고리 반영
**상태**: Draft

## 1. Problem Statement

많은 사람들이 "나는 카페를 좋아해"라고 말하지만, **정작 왜 좋아하는지, 그 안에 어떤 행위가 있는지는 모른다.** 이 막연함은 진로 선택, 취미 탐색, 직업 방향 설정 등에서 반복적인 혼란을 만든다.

자기발견에는 두 가지 경로가 있다. 이미 하는 것 속에서 "왜?"를 찾는 것, 그리고 **새로운 경험을 통해 몰랐던 나를 발견하는 것**. 기존 앱들은 이 두 경로를 모두 안내하는 서비스가 없다.

## 2. Goals

1. **사용자가 자신의 "진짜 동사"를 발견하는 경험을 제공한다** — 4주 내 최소 1개의 동사 발견
2. **귀찮은 사람도 지속할 수 있는 구조를 만든다** — 7일 유지율 40% 이상
3. **증거 기반 콘텐츠로 신뢰를 확보한다** — 모든 미션/질문에 학술 출처 표시
4. **안전한 앱을 만든다** — 우울증 환자 등 취약 사용자에게 위험한 정보를 주지 않는다
5. **MVP를 최소 리소스로 벌리드한다** — 1인 개발 기준 3개월 내 출시 가능한 범위

## 3. Non-Goals (MVP에서 제외)

1. **커뮤니티 기능** — 먼저 개인 경험을 완성한 후 고려
2. **온보딩 A/B 테스트** — MVP는 단일 플로우로 시작
3. **웨어러블/바이오 연동** — 기술적 복잡도 높음
4. **유료 구독 모델** — MVP는 완전 무료. PMF 확인 후
5. **다국어 지원** — 한국어 전용으로 시작

## 4. Target Users

### Primary: "막연한 탐색자" (20대 후반~30대)
자신이 뭘 좋아하는지 말할 수 있지만, 왜 좋아하는지는 모르는 사람. INFP 성향, 관심은 많지만 실행은 느린 타입.

### Secondary: "성장 갈망자"
자기계발에 관심 있지만 매번 똑같은 자기계발서만 읽다 말는 사람.

## 5. User Stories

- **미션 수행**: "나는 매일 간단한 행동 미션을 받아 실행하고 싶다. 이미 하는 것을 새로운 눈으로 보거나, 작은 새로운 경험을 해보고 싶다."
- **의미 이해**: "미션을 왜 하는지 알고 싶다. 이유를 알아야 동기가 생기니까."
- **기록**: "사진과 글로 느낀 걸 기록하고 싶다. 그림일기처럼 간단하게."
- **피드백**: "AI 담임선생님의 따뜻한 피드백을 받고 싶다."
- **패턴 발견**: "일요일에 한 주를 돌아보며 내 패턴을 발견하고 다음 주 방향을 정하고 싶다."
- **신뢰**: "미션의 근거가 되는 연구를 확인하고 싶다."

## 6. Requirements

### P0: Must-Have (MVP 필수)

#### 6.1 일일 미션 시스템
매일 아침 행동 미션 1개를 제공. **두 가지 카테고리:**

**🔍 관찰 미션 (경로 A)** — 이미 하는 것 속에서 발견하기
- 예: "오늘 가장 오래 머무른 장소에서 사진 찍기"

**🌱 탐색 미션 (경로 B)** — 작은 새로운 경험 해보기
- 예: "평소 안 가는 길로 출퇴근해보기", "편의점에서 한 번도 안 먹어본 거 하나 사먹기"
- 조건: 돈 안 들거나 소액, 30분 이내, 집 근처에서 가능

**공통 구조**: 행동 지시 + 의미(이유) + 출처
- MVP 기준: **관찰 미션 15개 + 탐색 미션 15개 + 성찰 질문 30개**
- 수락 검증: 모든 미션에 학술 출처(DOI) 연결

#### 6.2 그림일기 기록 UI
- 상단: 사진 촬영 또는 간단 그림 캔버스
- 하단: 텍스트 입력 (3줄 이내 권장, 최대 500자)
- 저장 후 수정 가능

#### 6.3 AI 담임선생님 피드백
- 톤: 초등학교 담임선생님이 빨간 펜으로 적어주는 느낌
- 구조: 공감 → 발견 포인트 → 다음 단계 힌트
- Layer 1 DB에 없는 내용은 생성 금지

#### 6.4 알림 시스템
- 기본값: 오전 8시(미션), 오후 2시(체크인), 오후 9시(기록)
- 일요일 오전 10시(한 주 정리)
- 사용자 커스터마이징 가능

#### 6.5 한 주 정리 (주간 리뷰)
- Part A (AI 자동): 타임라인, 감정 그래프, 키워드 클러스터, 동사 발견 리포트
- Part B-1 (~30초): Best Moment 1개 탭
- Part B-2 (~1분): 동사 확인 + 다음 주 방향 선택
- Closing: AI 주간 요약 피드백
- 전체 플로우 2분 이내

#### 6.6 안전 시스템 (Safety Guardrails)
- 3단계 위험 키워드 감지 (Level 1~3)
- Level 3 시 AI 즉시 중단 + 위기상담 연결 (1577-0199, 1393)
- AI 응답 금지 목록: 진단, 의료 조언, 근거 없는 낙관, 감정 축소

#### 6.7 출처 시스템
- [1][2] 각주 표시 → 탭하면 팝업
- 별도 출처 인덱스 페이지

### P1: Nice-to-Have

- **대시보드**: 동사 목록, 미션 완료율, 관찰/탐색 비율
- **성찰 질문**: 미션 후 선택적 질문, 탭으로 답변
- **오후 2시 체크인**: 미션 실행 리마인더
