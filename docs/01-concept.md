# 01. 컨셉 정리

## 핵심 정의

> 누구나 자신이 진짜 좋아하는 것을 발견할 수 있도록, 행동 미션과 가이드된 성찰 질문을 통해 자기 이해를 넓혀가는 앱

"나는 카페를 좋아해" → "나는 사실 '아무도 나를 모르는 곳에서 혼자이기'를 좋아하는 사람이었어"

이처럼 **좋아하는 것 뒤에 숨은 진짜 동사(verb)**를 발견하는 과정을 돕는다.

## 디자인 원칙

1. **귀찮은 사람도 할 수 있게** — 하루 미션 실행 5분 이내, 기록 3줄 이내
2. **부담 없이** — "잘 써야 한다"는 압박 제거. 사진 한 장 + 한 줄도 OK
3. **지속 가능하게** — 매일 다른 미션으로 신선함 유지, AI 피드백으로 보상감

## 대표 예시: 카페

| 단계 | 내용 |
|------|------|
| 표면 | "나는 카페를 좋아해" |
| 미션 | "오늘 카페 대신 공원 벤치에서 10분 앉아보기" |
| 기록 | "카페보다 덜 편했는데, 바람이 좋았다. 근데 혼자인 게 핵심이었던 것 같다" |
| 발견 | 카페가 좋은 게 아니라, **'아무도 나를 모르는 곳에서 혼자이기'**가 좋았던 것 |
| 동사 | **혼자이기** |

## 메시지 전략

### 핵심 표현
- "막연했던 것에 이름을 붙이는 과정"
- "내 안의 말을 만나는 시간"

### 전달 방식: Show, Don't Tell
직접 "생각을 글자로 꺼내는 행위가 중요합니다"라고 설명하지 않는다.
대신 사용자가 미션을 수행하고 기록하는 과정에서 자연스럽게 체험하게 한다.

### 터치포인트
- **온보딩**: 카페 예시로 "이런 경험 해본 적 있지 않아?" 공감 유도
- **주간 리뷰**: AI가 "이번 주에 이런 패턴이 보여요" → 자연스러운 언어화 경험
- **동사 발견 순간**: "당신이 진짜 좋아하는 건 '혼자이기'였어요" → 이름 붙이기

## 타겟 사용자

### Primary: "막연한 탐색자" (20대 후반~30대)
자신이 뭘 좋아하는지 말할 수 있지만, 왜 좋아하는지는 모르는 사람. INFP 성향, 관심은 많지만 실행은 느린 타입.

### Secondary: "성장 갈망자"
자기계발에 관심 있지만 매번 같은 자기계발서만 읽다 말는 사람.

## 차별화

| 기존 앱 | 접근 | unverb |
|---------|------|--------|
| 데일리오 | "오늘 기분이 어땠어?" | "오늘 이걸 해보고, 느낀 걸 적어봐" |
| 마보 | "마음을 관찰해봐" (듣기) | "행동해보고, 발견해봐" (하기) |
| 마인들링 | "멘탈을 케어합니다" (치료) | "진짜 좋아하는 걸 찾아봐" (발견) |
| AI 저널링 | "오늘 하루를 적어봐" (쓰기) | "이 미션을 해보고, 기록해봐" (행동+쓰기) |
